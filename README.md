# wallet-base

**Base library for wallet develop**

install
---

* `git clone https://github.com/trustnote/wallet-base.git`
* install [nodejs](nodejs.org)
* install grunt-cli

        `npm install -g grunt-cli`

How to use
---

#### nodejs

```
let core = require("wallet-base/lib")
```


#### Web

```
cd wallet-base
npm install
grunt
```

include `public/core.js`

see `demo/index.html`
